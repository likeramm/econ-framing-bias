# NLP 기반 경제뉴스 프레이밍 편향이 주가에 미치는 영향 분석 시스템

## Analyzing the Impact of Economic News Framing Bias on Stock Prices Using NLP

---

## 1. 프로젝트 개요

### 1.1 프로젝트 요약

한국 경제뉴스에서 동일한 경제 이벤트(GDP 발표, 기준금리 결정, 고용지표 발표 등)가 언론사별로 상이하게 보도되는 현상, 즉 **프레이밍 편향(Framing Bias)**을 NLP 모델로 자동 탐지하고, 이러한 편향이 **실제 금융시장(주가, 거래량)에 인과적 영향을 미치는지** 이벤트 스터디, 그랜저 인과관계 검정, 매개분석 등 다중 통계 기법으로 검증하는 데이터 기반 분석 시스템이다.

### 1.2 핵심 아이디어

본 시스템의 핵심은 **"같은 숫자, 다른 해석"** 현상의 정량화에 있다. 예를 들어, 한국은행이 "2분기 GDP 성장률 2.3%"를 발표했을 때:

- **A 언론사**: "견조한 성장세 지속, 하반기 반등 기대" (낙관적 프레이밍)
- **B 언론사**: "성장 둔화 우려 확대, 잠재성장률 하회" (비관적 프레이밍)
- **C 언론사**: "우려에도 불구하고 선방, 펀더멘털 견고" (방어적 프레이밍)
- **D 언론사**: "OECD 평균 상회, 미국 대비 양호" (비교적 프레이밍)

동일한 수치 하나가 언론사의 선택에 따라 전혀 다른 메시지로 전달된다. 본 프로젝트는 이 과정을 6가지 프레이밍 유형으로 분류하는 AI 모델을 구축하고, 편향 점수를 정량화한 뒤, 그 편향이 실제 관련 섹터 주가의 비정상 수익률(CAR)과 통계적으로 유의한 관계가 있는지를 검증한다.

### 1.3 핵심 흐름

```
경제 지표 발표 (GDP, 금리, 고용률 등)
       ↓
언론사별 기사 수집 (10개 언론사, 5년간, 6,381건)
       ↓
AI 모델 기반 프레이밍 분류 & 편향 점수 산출
  ┌─ KLUE-RoBERTa: 프레이밍 6유형 분류
  └─ KcELECTRA: 감성 강도 측정 (-1 ~ +1)
       ↓
┌──────────────────────┬───────────────────────┐
│  주가 반응 분석        │  소비심리지수 상관분석   │
│  (단기: 1~5일 CAR)    │  (중기: 월별 CCSI)      │
└──────────────────────┴───────────────────────┘
       ↓
"미디어 프레이밍 → 시장 심리 → 주가" 인과 경로 검증
       ↓
실시간 편향 탐지 웹 서비스 제공
```

### 1.4 프로젝트 동기 및 문제의식

**1) 미디어 프레이밍의 영향력 증대**

현대 투자자의 대다수는 경제 지표를 원시 데이터로 직접 확인하기보다 뉴스 기사를 통해 간접적으로 접한다. 이때 기사의 프레이밍, 즉 동일한 팩트를 어떤 맥락과 어조로 전달하느냐에 따라 독자의 경제 인식이 달라지고, 이는 투자 의사결정에까지 영향을 미칠 수 있다. 특히 한국의 경우 포털 뉴스(네이버) 중심의 소비 구조로 인해 헤드라인 프레이밍의 영향력이 더욱 크다.

**2) 기존 연구의 한계**

기존 미디어-주가 연구는 대부분 (1) 영어권 시장 대상, (2) 단순 감성(긍정/부정) 이진 분류, (3) 단일 언론사 분석에 머물러 있다. 그러나 실제 미디어 편향은 단순 감성보다 복잡한 프레이밍 구조를 가지며, 동일 이벤트에 대한 다수 언론사의 보도 차이를 비교해야 편향의 실체가 드러난다.

**3) 본 프로젝트의 접근**

본 프로젝트는 (1) 한국어 경제뉴스에 특화된 6가지 프레이밍 유형 분류 체계를 설계하고, (2) Transformer 기반 NLP 모델(KLUE-RoBERTa, KcELECTRA)로 자동 분류한 뒤, (3) 10개 언론사 간 편향 차이를 정량적 점수로 산출하고, (4) 이벤트 스터디 및 인과분석을 통해 편향이 주가에 미치는 실질적 영향을 검증한다. 나아가 편향이 소비자심리지수(CCSI)를 매개하여 간접적으로 주가에 영향을 미치는 경로까지 분석함으로써, 단순 상관관계를 넘어 인과적 메커니즘을 밝히고자 한다.

## 2. 학술적 배경 및 차별성

### 2.1 이론적 배경

**프레이밍 이론 (Framing Theory)**

Entman(1993)에 따르면 프레이밍이란 "인지된 현실의 특정 측면을 선택하여 부각시키는 것"으로, 동일한 정보라도 어떤 프레임으로 제시하느냐에 따라 수용자의 해석이 달라진다. 경제뉴스에서 이 효과는 특히 강력한데, 경제 지표는 본질적으로 다면적이어서(예: GDP 2.3%가 "성장"인지 "둔화"인지는 맥락에 따라 다름) 언론사의 프레이밍 선택이 독자의 경제 인식을 크게 좌우한다.

**미디어 감성과 주가 관계**

Tetlock(2007)은 월스트리트저널의 비관적 어조가 다우존스 지수 하락을 예측함을 보였고, Engelberg & Parsons(2011)는 같은 기업의 실적 발표에 대해 지역 신문의 보도 톤이 해당 지역 투자자의 거래 행태에 영향을 미침을 입증했다. 그러나 이들 연구는 모두 영어권 시장에 한정되며, 단순 감성(긍정/부정)만을 다루었다는 한계가 있다.

**행동경제학적 관점**

Kahneman & Tversky(1979)의 전망이론에 따르면 동일한 경제 성과도 이익 프레임과 손실 프레임으로 제시될 때 의사결정이 달라진다. 이를 미디어 맥락에 적용하면, 같은 GDP 수치를 "성장 지속"(이익 프레임)과 "둔화 우려"(손실 프레임)로 보도할 때 투자자의 반응이 비대칭적일 수 있다.

### 2.2 선행 연구와의 차별점

| 기존 연구 | 한계 | 본 프로젝트의 차별점 |
|-----------|------|---------------------|
| Tetlock(2007): 미디어 감성 → 주가 | 영어권, 단일 매체 | **한국어** 10개 언론사, 한국 시장 특화 |
| 일반적 감성분석 연구 | 긍/부정 이진 분류 | **6가지 프레이밍 유형** 다차원 분류 |
| 단일 언론사 기반 분석 | 편향 비교 불가 | **동일 이벤트 × 다중 언론사** 편향 점수 비교 |
| 감성 → 주가 직접 회귀 | 매개 경로 미검증 | 편향 → **소비심리(CCSI)** → 주가 **매개분석** |
| 1~2년 단기 분석 | 일반화 한계 | **5년** (2021~2026) 장기 데이터, 다양한 경제 국면 포함 |
| 단일 통계 기법 | 강건성 부족 | 이벤트스터디 + 그랜저 + 매개분석 + 패널회귀 **다중 검증** |

### 2.3 연구 질문 (Research Questions)

- **RQ1:** 동일 경제 지표에 대해 한국 언론사들의 프레이밍은 어떻게 다른가? 보수/진보/경제지/중립 언론사 그룹 간 체계적 차이가 존재하는가?
- **RQ2:** 언론사별 프레이밍 편향 점수는 시간에 따라 일관된 패턴을 보이는가? 즉, 특정 언론사가 지속적으로 낙관적 혹은 비관적 프레이밍을 유지하는가?
- **RQ3:** 부정적 프레이밍의 집중도가 높을수록 관련 섹터 주가의 비정상 수익률(CAR)이 음(-)의 방향으로 유의한가? 편향 분산(언론사 간 의견 불일치)이 클수록 거래량이 증가하는가?
- **RQ4:** 미디어 프레이밍이 소비심리지수(CCSI) 변화를 매개하여 간접적으로 주가에 영향을 미치는 경로가 통계적으로 유의한가?

### 2.4 핵심 가설

> **H1:** 동일 경제 이벤트에 대해 부정적 프레이밍 비율이 높을수록, 관련 섹터 ETF의 발표 후 1~5일 누적 비정상 수익률(CAR)이 음(-)의 방향으로 유의하다.

> **H2:** 편향 분산(언론사 간 의견 불일치)이 클수록 거래량이 증가한다.

> **H3:** 미디어 편향 점수 → CCSI 변화 → 주가 변동의 매개 경로에서 간접효과가 유의하다.

## 5. AI 분석 방법론

### 5.1 프레이밍 유형 분류 체계 (6가지)

| 유형 | 설명 | 예시 헤드라인 |
|------|------|-------------|
| **낙관적** (Optimistic) | 긍정적 전망 강조 | "견조한 성장세 지속, 하반기 반등 기대" |
| **비관적** (Pessimistic) | 부정적 전망 강조 | "성장 둔화 우려 확대, 불확실성 고조" |
| **경고적** (Alarmist) | 위기 상황 부각 | "경제 위기 신호, 금융위기 재현 우려" |
| **방어적** (Defensive) | 우려를 인정하되 완화 | "우려에도 불구하고 펀더멘털 견고" |
| **비교적** (Comparative) | 다른 국가/시기 비교 | "미국 대비 양호한 수준, OECD 평균 상회" |
| **중립적** (Neutral) | 사실 중심 보도 | "2분기 GDP 2.3% 기록, 전분기 대비 0.1%p 상승" |

### 5.2 편향 점수 산출 공식

```
Bias Score (언론사 i, 이벤트 j) =
    α × framing_type_score     (프레이밍 유형 점수: -2 ~ +2)
  + β × sentiment_intensity    (감성 강도: -1 ~ +1)
  + γ × keyword_polarity       (핵심 키워드 극성)
  ─────────────────────────────────────────
  → 최종 편향 점수: -3 (극부정) ~ +3 (극긍정)

가중치:
  α = 0.40 (프레이밍 유형)
  β = 0.35 (감성 강도)
  γ = 0.25 (키워드 극성)
```

**프레이밍 유형별 점수 매핑:**

| 유형 | 점수 |
|------|------|
| 낙관적 (Optimistic) | +2 |
| 방어적 (Defensive) | +1 |
| 중립적 (Neutral) | 0 |
| 비교적 (Comparative) | 0 |
| 비관적 (Pessimistic) | -1 |
| 경고적 (Alarmist) | -2 |

**편향 분산:**

```
편향 분산 (이벤트 j) = Var(Bias Scores across all media)
→ 높을수록 언론사 간 의견 분열이 큰 이벤트
```

### 5.3 AI 모델 구성

| 용도 | 모델 | 입력 | 출력 |
|------|------|------|------|
| 프레이밍 분류 | KLUE-RoBERTa-large (Fine-tuned) | 기사 본문 | 6가지 프레이밍 유형 |
| 감성 강도 분석 | KcELECTRA-base | 기사 본문 | 감성 점수 (-1 ~ +1) |
| 키워드 추출 | KeyBERT (Korean) | 기사 본문 | 프레이밍 근거 키워드 |
| 이벤트 매칭 | Sentence-BERT (ko) | 기사 제목 | 경제 이벤트 자동 매칭 |

### 5.4 통계 분석 방법

| 분석 | 목적 | 방법론 |
|------|------|--------|
| **이벤트 스터디** | 경제 이벤트 발표 전후 주가 비정상 수익률 측정 | CAR(-1, +5) 산출, 추정기간 [-120, -11], 이벤트 윈도우 [-10, +10] |
| **그랜저 인과관계** | 편향 점수가 주가 변동을 시차 예측하는지 검정 | statsmodels, 시차 1~10일 |
| **매개 분석** | 편향 → 소비심리(CCSI) → 주가 경로 검증 | pingouin, 직접/간접 효과 분리 |
| **패널 회귀분석** | 언론사 × 시간 패널 데이터 다변량 분석 | linearmodels, 고정/랜덤 효과 |

**이벤트 스터디 윈도우:**

```
      추정 기간                이벤트 윈도우
◄─────────────────►    ◄───────────────────►
[-120일    -11일]      [-10일  발표일  +10일]
                              ↑
                        경제 지표 발표

- 추정 기간: 정상 수익률 모델 학습 (시장 모형)
- 이벤트 윈도우: 비정상 수익률(AR) 측정
- CAR(-1,+5): 발표 전일~후5일 누적 비정상 수익률
```
