# 경제 뉴스 프레이밍 편향 탐지기 + 주가 영향 분석

## 프로젝트 확장 컨셉

**기존 아이디어:** 동일 경제 지표 → 언론사별 프레이밍 차이 분석
**확장 아이디어:** 프레이밍 편향이 실제 주가에 영향을 미치는가?

```
경제 지표 발표 (GDP, 금리, 고용률 등)
       ↓
언론사별 기사 수집 (10개+)
       ↓
프레이밍 분류 & 편향 점수 산출
       ↓
┌─────────────────────┬──────────────────────┐
│  주가 반응 분석      │  소비심리지수 상관분석  │
│  (단기: 1~5일)      │  (중기: 월별)          │
└─────────────────────┴──────────────────────┘
       ↓
"미디어 프레이밍 → 시장 심리 → 주가" 인과 경로 검증
```

---

## 왜 이게 학술적으로 가치 있는가

### 선행 연구와의 차별점

| 기존 연구 | 이 프로젝트의 차별점 |
|-----------|-------------------|
| 영어권 미디어 감성 → 주가 (Tetlock, 2007) | **한국어** 미디어, 한국 시장 특화 |
| 단순 감성(긍/부정) 분석 | **프레이밍 유형** 분류 (낙관/비관/경고/중립/방어적 등) |
| 단일 언론사 | **동일 이벤트, 다중 언론사** 비교 → 편향 점수화 |
| 감성 → 주가 직접 연결 | 미디어 편향 → 소비심리 → 주가의 **매개 경로** 검증 |

### 연구 질문 (Research Questions)

- **RQ1:** 동일 경제 지표에 대해 한국 언론사들의 프레이밍은 어떻게 다른가?
- **RQ2:** 언론사별 프레이밍 편향 점수는 시간에 따라 일관된 패턴을 보이는가?
- **RQ3:** 부정적 프레이밍의 집중도가 높을수록 관련 섹터 주가 하락 폭이 큰가?
- **RQ4:** 미디어 프레이밍이 소비심리지수(CCSI) 변화를 매개하여 주가에 영향을 미치는가?

## 기술 스택 상세

### 데이터 수집

| 데이터 | 소스 | 방법 | 비고 |
|--------|------|------|------|
| 뉴스 기사 | 네이버뉴스 | BeautifulSoup + requests | 경제 섹션, 언론사별 필터링 |
| 경제 지표 | 한국은행 ECOS API | REST API (무료) | GDP, CPI, 금리, 고용률 등 |
| 주가 데이터 | FinanceDataReader / pykrx | Python 라이브러리 | KOSPI, KOSDAQ, 섹터별 |
| 소비심리지수 | 한국은행 ECOS API | REST API | CCSI 월별 데이터 |

### 모델

| 용도 | 모델 | 설명 |
|------|------|------|
| 프레이밍 분류 | KLUE-RoBERTa-large | 6가지 프레이밍 유형 분류 (아래 참고) |
| 감성 강도 분석 | KcELECTRA-base | 감성 점수 (-1 ~ +1) 연속값 |
| 키워드 추출 | KeyBERT (Korean) | 프레이밍 근거 키워드 |
| 토픽 매칭 | Sentence-BERT (ko) | 기사 ↔ 경제 이벤트 자동 매칭 |

### 프레이밍 유형 분류 체계 (6가지)

```
1. 낙관적 (Optimistic)     "견조한 성장세 지속"
2. 비관적 (Pessimistic)    "성장 둔화 우려 확대"
3. 경고적 (Alarmist)       "경제 위기 신호"
4. 방어적 (Defensive)      "우려에도 불구하고..."
5. 비교적 (Comparative)    "미국 대비 양호한 수준"
6. 중립적 (Neutral)        "2분기 GDP 2.3% 기록"
```

### 분석 방법론

| 분석 | 라이브러리 | 목적 |
|------|-----------|------|
| 이벤트 스터디 | eventstudy (Python) | 지표 발표일 전후 주가 비정상 수익률(CAR) |
| 그랜저 인과관계 | statsmodels | 편향 점수 → 주가 시차 인과관계 |
| 매개 분석 | pingouin / lavaan (R) | 편향 → CCSI → 주가 매개 경로 |
| 패널 회귀 | linearmodels (Python) | 언론사 × 시간 패널 데이터 |
| 시각화 | Plotly + Streamlit | 인터랙티브 대시보드 |

## 편향 점수 산출 공식

```
Bias Score (언론사 i, 이벤트 j) =
    α × framing_type_score    (프레이밍 유형: -2 ~ +2)
  + β × sentiment_intensity   (감성 강도: -1 ~ +1)
  + γ × keyword_polarity      (핵심 키워드 극성)
  ─────────────────────────────────────────
  → 최종 편향 점수: -3 (극부정) ~ +3 (극긍정)

편향 분산 (이벤트 j) =
    Var(Bias Scores across all media)
    → 높을수록 언론사 간 의견 분열이 큰 이벤트
```

## 주가 분석 확장 설계

### 핵심 가설

> **H1:** 동일 경제 이벤트에 대해 부정적 프레이밍 비율이 높을수록,
> 관련 섹터 ETF의 발표 후 1~5일 누적 비정상 수익률(CAR)이 음(-)의 방향으로 유의하다.

> **H2:** 편향 분산(언론사 간 의견 불일치)이 클수록 거래량이 증가한다.

> **H3:** 미디어 편향 점수 → CCSI 변화 → 주가 변동의 매개 경로가 유의하다.

### 이벤트-섹터 매핑

```python
EVENT_SECTOR_MAP = {
    "GDP_성장률":     ["KOSPI", "KODEX200"],
    "기준금리":       ["은행", "보험", "건설", "부동산"],
    "소비자물가":     ["유통", "식품", "내수소비"],
    "수출입동향":     ["반도체", "자동차", "조선"],
    "고용동향":       ["서비스", "내수소비"],
    "부동산가격":     ["건설", "부동산", "은행"],
}
```

### 이벤트 스터디 윈도우

```
        추정 기간              이벤트 윈도우
  ◄─────────────────►  ◄───────────────────►
  [-120일    -11일]    [-10일  발표일  +10일]
                              ↑
                         경제 지표 발표

  - 추정 기간: 정상 수익률 모델 학습 (시장 모형)
  - 이벤트 윈도우: 비정상 수익률(AR) 측정
  - CAR(-1,+5): 발표 전일~후5일 누적 비정상 수익률
```

## 개발 로드맵

### Phase 1: 데이터 기반 구축 (4~6주)

**목표:** 최소 6개월치 뉴스-지표-주가 페어링 데이터셋 구축

- [ ] 네이버뉴스 크롤러 (10개 언론사, 경제 섹션)
- [ ] ECOS API 연동 (주요 경제 지표 6종)
- [ ] 주가 데이터 수집 파이프라인 (pykrx / FinanceDataReader)
- [ ] 기사-이벤트 자동 매칭 모듈
- [ ] 데이터 정제 및 EDA

**산출물:** `data/` 디렉토리에 정제된 데이터셋

### Phase 2: 프레이밍 분류 모델 (3~4주)

**목표:** F1 ≥ 0.80 이상의 프레이밍 분류기

- [ ] 학습 데이터 라벨링 (500+ 기사, 6가지 프레이밍)
- [ ] KLUE-RoBERTa Fine-tuning
- [ ] KcELECTRA 감성 강도 모델
- [ ] 편향 점수 산출 모듈
- [ ] 모델 성능 평가 및 오류 분석

**산출물:** 학습된 모델 체크포인트, 평가 리포트

### Phase 3: 주가 상관분석 (3~4주)

**목표:** 프레이밍 편향 ↔ 주가 관계 통계적 검증

- [ ] 이벤트 스터디 구현 (CAR 계산)
- [ ] 그랜저 인과관계 검정
- [ ] 매개 분석 (편향 → CCSI → 주가)
- [ ] 패널 회귀분석
- [ ] 통계 결과 정리

**산출물:** 분석 노트북, 통계 결과 테이블

### Phase 4: 시각화 & 대시보드 (2~3주)

**목표:** 포트폴리오용 인터랙티브 데모

- [ ] Streamlit 대시보드 설계
- [ ] 언론사별 편향 프로파일 시각화
- [ ] 이벤트 타임라인 + 주가 오버레이
- [ ] 실시간 분석 데모 (새 기사 입력 → 편향 예측)
- [ ] 배포 (Streamlit Cloud / HuggingFace Spaces)

**산출물:** 배포된 웹 대시보드

### Phase 5: 논문화 & 고도화 (2~4주)

**목표:** 대학원 지원용 연구 보고서 or 학회 투고

- [ ] 관련 선행연구 정리 (Literature Review)
- [ ] 방법론 섹션 작성
- [ ] 결과 해석 및 시사점 도출
- [ ] 한계점 및 향후 연구 방향
- [ ] 한국정보과학회(KCC) / 한국언론학회 등 학회 투고 검토

**산출물:** 연구 보고서 (8~15페이지)
